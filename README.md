# 🌟 Friendship Day Celebration Website\n\nA beautiful, interactive website to celebrate the bonds of friendship!\n\n## ✨ Features\n\n- 🎨 **Stunning Design**: Warm, cheerful color palette with smooth animations\n- 📸 **Memory Gallery**: Beautiful photo gallery with hover effects\n- 💬 **Inspirational Quotes**: Curated friendship quotes to warm the heart\n- 🧪 **Friendship Tester**: Fun and interactive friendship bond calculator\n- 🎉 **Confetti Effects**: Celebratory animations throughout\n- 🎵 **Sound Effects**: Interactive audio feedback with Tone.js\n- ✨ **Magical Cursor**: Trailing emoji particles as you move your cursor\n- 📱 **Responsive**: Works perfectly on all devices\n\n## 🚀 Live Demo\n\nVisit: [https://iitdelhi871-cloud.github.io/friendship-day/](https://iitdelhi871-cloud.github.io/friendship-day/)\n\n## 📂 Project Structure\n\n```\n.\n├── index.html       # Main HTML structure\n├── style.css        # All styling and animations\n├── script.js        # Interactive features and animations\n└── README.md        # This file\n```\n\n## 🛠️ Technologies Used\n\n- **HTML5**: Semantic markup\n- **CSS3**: Flexbox, Grid, animations, and gradients\n- **Vanilla JavaScript**: Interactive features\n- **Canvas Confetti**: Confetti effects library\n- **Tone.js**: Web Audio API for sound effects\n- **Google Fonts**: Caveat and Poppins fonts\n\n## 📋 Sections\n\n1. **Home** - Hero section with a welcoming message\n2. **Quotes** - Inspirational friendship quotes\n3. **Gallery** - Memory photos with Polaroid-style effects\n4. **Friendship Tester** - Fun bond strength calculator\n5. **Surprise** - Random quote generator with animations\n\n## 🎯 How to Use\n\n1. Open the website in your browser\n2. Enjoy the animations and interactive features\n3. Scroll through memories and quotes\n4. Test your friendship bond with a name\n5. Generate surprise messages\n\n## 💡 Customization\n\nTo customize:\n- **Edit photos**: Update image URLs in the gallery section of `index.html`\n- **Change colors**: Modify CSS variables in `style.css` (`:root` section)\n- **Add quotes**: Add more quotes to the `quotes` array in `script.js`\n\n## 📝 License\n\nMade with ❤️ for Friendship Day\n\n---\n\n**Happy Friendship Day!
